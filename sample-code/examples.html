Examples

1: Pride & Prejudice

CiteIt.net allows inline popup quotations use the <q> tag:

In Chapter 5 of Pride and Prejudice, Elizabeth confesses how Darcy offended her, saying: I could easily forgive his pride, if he had not mortified mine.

2: Tyler Cowen interviews David Wolpe

In a medium.com article, Tyler Cowen commented to David Wolpe about how social media strips away context:
I think that Judaism has the same problem that any thick civilization has in a world in which, as you say, context is stripped away. And not only is context stripped away, but attention to any one thing is scanter and less than it used to be.

You can mark up the quote with html tags such as <ul>, <li>, <b> or any other tag that will not change the text version of the quote when the hmtl is converted to text.
